# 🎀 有価証券報告書のきらきら保管庫 (annual) ✨

**タイトル**: EDINET 提出書類のひみつ基地 (annual)
**お仕事の目的**: たくさんの会社の「有価証券報告書」を、会社ごとにきれいに整頓して大切に保管することだよっ！✨
**解決したいお悩み**: 膨大なデータがバラバラにならないように、AIさんがいつでも解析できるように可愛く整理しておきたいのっ！💦

## エグゼクティブサマリー
この場所は、EDINETから集めてきた大事な「有価証券報告書」たちが、会社ごとの「おうち（フォルダ）」に仲良く住んでいる特別なアーカイブだよっ！🏠✨ メタデータ（JSON）や中身を解析したデータ（TSV）、そして元のPDFがぜーんぶ揃っているから、アルファを見つけるための「知恵の宝石」がいつでも取り出せちゃうんだぁ！🎁💎 データの整理整頓をしっかりして、最強のクオンツシステムを支えちゃうよっ！💖

---

## 🌟 おうち（フォルダ）の中身はどうなってるの？
ここは、EDINETから取ってきた「有価証券報告書」たちが、会社ごとに大切にしまわれている場所だよっ！✨
EDINETコード（Eからはじまる名前）のフォルダの中に、宝物が詰まってるんだぁ 🎁

- **`EXXXXX/`** 📁
  - 各会社さん専用の特別なフォルダだよっ！
  - その中には、JSON形式のメタデータや、中身をかしこく解析したTSVファイル、そして元のPDFファイルが大切にしまわれてるんだぁ 📄🔍
