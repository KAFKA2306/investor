# 🎀 わるいことしちゃダメ！会計不正検知プロジェクト 🕵️‍♀️✨

**タイトル**: 会計不正検知タスクの準備 (Accounting Fraud Detection Task Prep)
**お仕事の目的**: 企業の報告書に隠された「粉飾」や「うそ」を暴く、正義のAIさんを育てるための最強データセットを作ることだよっ！
**解決したいお悩み**: 「どの報告書が怪しいの？」「どうやって不正の証拠を見つければいいの？」っていう難しい問題を、Claudeちゃんと一緒に解決したいんだぁ！( ๑`^´๑)

## エグゼクティブサマリー
このプロジェクトは、有価証券報告書の訂正内容をClaudeさん（LLM）が厳しくチェックして、不適切な会計処理をパトロールする探偵さんみたいなシステムだよっ！🕵️✨ 悪いことをしたデータと、きらきら真っ白なデータを上手に混ぜて、AIさんが「これはダメ！」って賢く判断できるように特訓の準備を整えちゃうんだからねっ！💖

---

## 🌟 どんなお仕事をしているの？

- **`prepare_fraud.py`** 🔍
  - `Claude`さん（LLM）にお願いして、訂正有価証券報告書の中から「これ、絶対おかしいよぉ…」っていう「悪い修正」を一生懸命探してもらうよ！
  - 「不適切会計」とかの怪しいキーワードを、`Claude`さんの鋭い目（アルゴリズム）でしっかりパトロールしちゃうんだからねっ！🚨✨

- **`prepare_nonfraud.py`** ✅
  - 不正なんて全然していない、きらきら真っ白で健全な「普通」の会社のデータをいっぱい集めてくるよ！
  - 悪い子を見分けるには、良い子を知ることも大事なんだもんっ！✨

- **`prepare_dataset.py`** 📦
  - 悪いデータと良いデータをまぜまぜして、AIさんが「これは不正！」「これはOK！」って特訓するための、最高に使いやすいデータセットをパッキングしちゃうよっ！🏋️‍♀️✨

- **`analyze_fraud_explanation.py`** 📖
  - `Claude`さんが「ここが怪しいんだよぉ…」って教えてくれた理由を、じっくりコトコト分析して、不正のパターンをみんなでお勉強するよっ！🧠📚
