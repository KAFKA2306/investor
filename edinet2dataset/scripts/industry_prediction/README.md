# 🎀 業種予測クイズ！AIさんのためのデータ工場 🏭 ✨
**タイトル**: 業種予測タスクのデータ準備
**お仕事の目的**: 有価証券報告書を読んで「この会社は、何屋さんかな？」って当てるための、最高に使いやすい学習データを作ることだよっ！
**解決したいお悩み**: 33業種もあって複雑すぎるし、データの数もバラバラでAIさんが困っちゃうのを、綺麗に整理して解決するよっ！✨

## エグゼクティブサマリー
AIさんが有価証券報告書を読んで、会社の業種をピタリと言い当てられるように、とびきり素敵なデータセットを準備するよ！💖 日本の33業種をAIさんが理解しやすい16種類にまとめ直して、各業種からバランスよく35社ずつピックアップ！「事業の状況」などのテキストをAIさんに読ませて、クイズ正解率100%を目指すための大事な準備なんだよ〜！🚀✨

---

「この会社は、何屋さんかな？(๑•́ ₃ •̀๑)」
有価証券報告書を読んで、会社のお仕事を当てるクイズ用データを作る場所だよっ！✨

## 🌟 どんな魔法を使っているの？ ✨

- **`prepare_dataset.py`** 📊
  - 日本の複雑な33種類の業種を、もっとわかりやすくて使いやすい「16種類」にギュギュッとまとめ直しちゃうよ！✨
  - 報告書を読み込んで、特定の業種に偏らないように各業種から「35社」ずつ、仲良くバランスよく集めるんだぁ 🧺💖
  - 会社の自己紹介（「事業の状況」とかの大事なページ！）をAIさんにたっぷり読ませて、業種当てクイズの正解を目指して特訓するよっ 🧠📚
