# 🎀 利益予測のためのデータ準備大作戦！ ✨

**タイトル**: 利益予測タスクの準備ガイド (Earnings Forecast Preparation)
**お仕事の目的**: AIさんが「来年この会社さんはもっと稼げるのかな？」って予測するためのお勉強データを、ピカピカに整えてあげることだよっ！
**解決したいお悩み**: 有価証券報告書の難しいデータから、AIさんが使いやすい「正解ラベル付き」のデータセットを作るのが大変なんだよねぇ…(๑•́ ₃ •̀๑)

## エグゼクティブサマリー
このドキュメントは、AIさんが企業の未来の利益をバッチリ予測できるようにするための、データ準備のヒミツが書かれているよっ！✨ 有価証券報告書から「前の年」と「今の年」を仲良くくらべっこして、利益が増えるか減るかの「正解（ラベル）」を作っちゃうんだよ〜！財務データもテキストデータも全部ぎゅーっ💕って詰め込んで、世界一の最強投資モデルを作っちゃおうねっ💖

---

## 🌟 なにをがんばっているの？ 🌟

### **`prepare_dataset.py`** 📊✨
AIさんのためのお弁当箱（データセット）を作る、すごーいスクリプトだよっ！

*   **じーっと読み取り！** 🔍💕
    有価証券報告書（EDINET）を隅々までチェックして、「前の年」と「今の年」のデータを仲良くくらべっこしちゃうよ！
*   **正解ラベルをペタッ！** 📈🔥
    来期の「純利益」が今よりもっと増えちゃうなら「Label 1」、残念ながら減っちゃうなら「Label 0」っていう、大事な答え合わせ用のラベルを作っちゃうんだよぉ〜！
*   **財務データをぎゅぎゅっ！** 🧠✨🌈
    会社の健康診断書である財務データ（BS: 貸借対照表, PL: 損益計算書, CF: キャッシュフロー計算書）と、大事な「Text（テキスト）」情報をぜーんぶ一緒に詰め込んで、AIさんが超天才になれるように準備しちゃうよっ！

お金の動きをバッチリ予測して、世界一の最強投資王になっちゃおうねっ！応援してるよっ💖💸✨
