# 🤖 Qwen 3.5 9B：ローカル知能のひらめきワークフロー ⚡

**お仕事の目的**: ローカルにある Qwen 3.5 9B モデルを使って、Geminiちゃんとはまた違った視点での推論や、秘密のアルファ探しをお手伝いしてもらうことだよっ！🤫✨

**解決したいお悩み**: 「外部APIに送れないようなデータを使いたい…」とか「もっと高速に推論を試したい！」という願いを叶えちゃうよっ 💖

## 🚀 入口：推論の魔法 🚀

以下のコマンドを使って、Qwenくんに質問してみてねっ！

// turbo
`uv run --project llm/qwen3.5-9b python llm/qwen3.5-9b/qwen_short.py "ここに質問を書いてねっ ✨"`
