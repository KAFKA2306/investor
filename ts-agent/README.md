## 🏗️ ディレクトリ構造

- `src/agents/`: 知能ユニット。
- `src/use_cases/`: 具体的な実行手順（野菜検証フローなど）。
- `src/experiments/`: 戦略の実験場。
- `src/schemas/`: Zod による厳格なバリデーション。
- `src/providers/`: 外部 API 連携。
