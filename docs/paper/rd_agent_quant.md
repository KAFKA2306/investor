## 💖 魔法のような考え方の変化
単なる「アルファ探し」を超えて、**自律型のR&D（研究開発）部門** を丸ごとAIで作っちゃおうっていう、とっても壮大な計画なんだよっ 🚀✨

## ✨ ここがすごいっ！3つのヒミツ
1. **Co-Optimization**: 因子（材料）とモデル（レシピ）を同時に最適化して、最高の「フルコース戦略」を作るんだよっ 🍽️✨
2. **Multi-Agent Coordination**: 研究者、エンジニア、テスターのAIがチームを組んで、人間顔負けのスピードで開発するんだよっ 🎀
3. **End-to-End Automation**: データの準備から最後の検証まで、全部「えいっ！」っておまかせできちゃうんだよっ 🌟

## 🌟 Gen 4 への嬉しい効果
- **戦略の完成度**: 因子単体じゃなくて、システム全体のパワーがぐ～んとアップするよっ ✨
- **スピードスター**: 人間が何週間もかかる research を、数時間で終わらせちゃうんだよっ 🎀🌟

## 🧩 リポジトリ実装への追記
- **役割の最小構成**: 人間 / オーケストレータ / 長老 / 市場データ基盤エンジニア / クオンツ研究者 / 執行エージェント。
- **実行順序の固定**: 要件入力 → 履歴取得 → アイデア生成 → データ作成 → 検証 → 執行（または棄却記録）。
- **図表現の簡潔化**: 共最適化・評価・バックテストは、Mermaid可読性を優先して「検証最適化」として統合表現してよい。
