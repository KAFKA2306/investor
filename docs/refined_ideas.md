## 3. 分析・監視のしかたっ！
- **JQuants API V2**: 株価や財務情報、Tickデータも分足も思いのままっ！細かい動きも見逃さないよっ✨
- **yfinance**: まずはお試しで株価や財務を知りたいときに、無料で使える心強い味方だよっ♪
- **EDINET API**: 公開買付や大量保有報告、自社株買いなどのイベントスタディに必須っ！無料でイベントを追いかけようねっ👀
- **e-STAT API**: 行政がまとめる景気動向や生産指数などの統計データをゲッチュ！幅広いデータで市場を俯瞰しちゃおうっ✨
- **auカブコム証券 API**: 発注を自動化して、チャンスを逃さず成就させるための秘密の武器だよっ！🚀
