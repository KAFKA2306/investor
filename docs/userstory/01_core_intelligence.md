# ユーザーストーリー：コア・インテリジェンス（稼働中のエージェント）

## LES (Large-scale Stock Forecasting)
- **クオンツ開発者として**、**LES フレームワーク**（ArXiv:2409.06289）を活用して LLM によるアルファ因子の動的生成を行いたい。これにより、手動のフィーチャーエンジニアリングなしで市場の変化に適応できる。
- **システムアーキテクトとして**、`LesAgent` が複数のサブエージェントを協調させ、動的な重み付けを行うことで、堅牢なシグナル生成を実現したい。

## PEAD (Post-Earnings Announcement Drift)
- **トレーダーとして**、**Hybrid PEAD** 戦略を用いて決算発表後の株価ドリフトを捉えたい。これにより、ファンダメンタルズの変化に対する市場の過小反応から利益を得られる。
- **リスク管理者として**、`PeadAgent` で財務上のサプライズとニュースのセンチメント解析をクロスバリデーションし、ダマシの少ないシグナルを抽出したい。

## ソーシャル解析 (X Intelligence)
- **モメンタムトレーダーとして**、**X (Twitter) のセンチメント**とトレンドの熱量を定量化したい。これにより、個人投資家主導の動きや「ミーム銘柄」の予兆を早期に検知できる。
- **データエンジニアとして**、特定のティッカー関連キーワードやインフルエンサーの影響力を監視し、リアルタイムの「ソーシャル・ヒートマップ」を提供したい。
