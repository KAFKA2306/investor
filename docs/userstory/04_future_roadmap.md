# ユーザーストーリー：将来のロードマップ（未実装・有用な機能）

## 1. まだできていないこと (Pending Features)
- **クオンツ開発者として**、複数の銘柄を組み合わせた**ポートフォリオ最適化機能**を実装したい。これにより、単一銘柄のアルファだけでなく、全体のボラティリティを抑えた運用が可能になる。
- **リスク管理者として**、市場の急変を検知して自動でポジションを解消する**セーフティ・サーキットブレーカー**を導入したい。現在は分析とスコアリングまでが主目的であり、積極的な防衛策が必要である。
- **データエンジニアとして**、JQuants 以外のデータソース（マクロ経済指標、オルタナティブデータなど）の自動取り込みパイプラインを完備したい。

## 2. 有ったら役立つこと (Nice to Have / Experimental)
- **投資家として**、AIがなぜその意思決定（スコア）を出したのかを自然言語で説明する**「説明可能なAI (XAI)」機能**が欲しい。ダッシュボードに「根拠の要約」を表示することで、信頼性を向上させる。
- **トレーダーとして**、過去の自分の予測精度を自己採点し、モデルの重みを自動調整する**自己進化型フィードバックループ**が欲しい。
- **システム管理者として**、異常検知やパフォーマンス低下を Slack や LINE に通知する**リアルタイムアラート機能**が欲しい。
- **UI/UX デザイナーとして**、過去のシミュレーション結果をインタラクティブに操作できる**高度なチャート機能**をダッシュボードに追加し、「Wowed」な体験をさらに高めたい。
