# 将来の計画：進展と拡張

## 1. まだできていないこと
- **ポートフォリオ全体の最適化**: 1つの銘柄で稼ぐだけでなく、複数の銘柄を組み合わせて、システム全体でのリスクを抑えながら安定して稼ぐ機能を検討しています。

## 2. あったら便利な機能

### 2.1 見やすい表示画面 (ダッシュボード)
- **新しい分析の自動表示**: 新しい投資戦略を追加したとき、画面のプログラムを書き換えなくても自動でグラフや表が表示される仕組みを作りたいと考えています。
- **思考プロセスの見える化**: AI がなぜその銘柄を選んだのか、その理由を詳しく辿れる機能を追加します。
- **レポートの自動作成**: 研究や検証が終わったら、自動的に報告書（Markdown 形式）を作成して保存されるようにします。
