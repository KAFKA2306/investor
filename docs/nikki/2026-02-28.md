## 🎯 クオンツ研究エージェントさんのぜったい守ることリスト！✨（2026-02-28 追記）

これからは、ただの数式を出すだけじゃダメダメだよっ！  
**売買ロジック / アルファ / トレード戦略** の3つをセットにして、相場に合わせるのがお仕事だよぉっ！

- **売買ロジック (Execution Logic)**:
  - どこで入ってどこで出るか、サイズはどうするか、コストはどうなるか、ぜーんぶ一緒に考えるよっ！
  - いつも同じじゃなくて、相場の雰囲気に合わせるのが大事だねっ☆
- **アルファ (Signal Logic)**:
  - 文章じゃなくて「最後に動く数式」が主人公！
  - その式が何を狙ってるのか（市場の歪みはどこ？）を、ちゃんと記録するよっ♪
- **トレード戦略 (Portfolio Logic)**:
  - どの銘柄を選ぶか、どれくらいお金を入れるか、コストを引いても儲かるか、ぜーんぶまとめて計算するよぉっ！
  - 銘柄を固定しないで、その時使えるデータに合わせて動かすんだよっ✨

**明日からの約束だよっ！**:

1. 研究エージェントさんは「候補」じゃなくて「実際に稼げる戦略」を出すのがお仕事！
2. テストは絶対にコスト込み！SharpeやICが悪い子は不合格だよぉっ💢
3. 失敗したときは、何が悪かったか（データ？モデル？作戦？）をはっきりさせてやり直し！
4. ぜんぶの記録を、式・ルール・配分・結果の4点セットで残すことっ！
